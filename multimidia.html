Multimídias

Multimídia na web é todo som, música, vídeo, filmes e animações que podem ser disponibilizadas para os usuários através de uma página HTML.

Imagem

A tag <img> é usada para incorporar uma imagem em uma página HTML. O atributo src especifica o caminho para a imagem a ser exibida. Existem duas maneiras de especificar o URL no atributo src: URL absoluto e URL relativo:

Código:

Absoluto: <img src ="https://exemplo.com/borboleta.jpeg">
Relativo: <img src ="img/borboleta.jpeg width ="300" height = "200">

Vídeo

A tag vídeo tem atributo de altura e largura, o controls informa que eu quero que os controles de nagevação do vídeo sejam exibidos. Esses dois "source" aparecem para se caso um tipo de vídeo não seja suportado pelo navegador, o navegador vai automaticamente tentar executar o segundo tipo. O primeiro é o tipo padrão e o segundo é um backup. Em um navegador moderno todas as extensões são executadas. Mas caso ocorra alguma coisa como se o link estiver quebrado e não executar o primeiro, ele irá executar o segundo. Então, se não executar nenhum dos dois "source", a mensagem será mostrada.

Código:

<video width="320" height="240" controls>
..<source src="video/video-gatinha.mp4" type="video/mp4">
..<source src="video/video-gatinha.ogg" type="video/ogg">
..Seu navegador não suporta a tag vídeo.
</video>

Vídeo do Youtube

Converter vídeos em formatos diferentes pode ser difícil e demorado Uma solução mais fácil é permitir que o YouTube reproduza os vídeos em sua página da web

Código:

<iframe
..width="560"
..height="315"
..src="https://www.youtube.com/embed/dlNtclPMMuE"
..title="YouTube video player"
..frameborder="0"
..allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
..allowfullscreen>
</iframe>

Áudio

Esse elemento é usado para reproduzir um arquivo de áudio em uma página da web.

Código:

<audio controls>
..<source src ="audio/som-golfinho.mp3" type ="audio/mp3">
..<source src ="audio/som-golfinho.ogg" type ="audio/ogg">
..Seu navegador não suporta o elemento áudio
</audio>
