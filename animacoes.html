Animações


Uma animação permite que um elemento mude gradualmente de um estilo para outro. Para usar animação, devemos primeiro especificar alguns quadros-chave (@keyframes). Os keyframes contêm os estilos que o elemento terá em determinados momentos.

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation

A regra keyframe

Quando especificamos estilos CSS dentro da regra @keyframes, a animação mudará gradualmente do estilo atua l para o novo estilo em determinados momentos. Para fazer uma animação funcionar, devemos vincular a animação a um elemento.

O exemplo a seguir vincula a animação exemplo ao elemento div. A animação durará 2 segundos e mudará gradualmente a cor de fundo do elemento div de cinza para amarelo.

No meu arquivo de estilos (style.css nesse caso)
OU
no HEAD do meu arquivo do navegador usando a tag <style>

@keyframes exemplo1 {
..from{
....background-color: silver;
..}
..to{
....background-color: yellow;
..}
}
.animacao1 {
width: 150px;
height: 150px;
background-color: silver;
animation-name: exemplo1;
animation-duration: 10s;
animation-delay: 2s;
animation-iteration-count: infinite;
}

No BODY do meu arquivo do navegador

<div class="animacao1"></div>

No exemplo anterior, especificamos quando o estilo será alterado usando as palavras chave "from" e "to"- que representa 0%(início) e 100%(completo).

Obs: a propriedade animation-duration define quanto tempo uma animação deve levar para ser concluída. Se esta propriedade não for especificada, nenhuma animação ocorrerá.

Também é possível usar por cento. Usando porcentagem, você pode adicionar quantas mudanças de estilo desejar.

No meu arquivo de estilos (style.css nesse caso)
OU
no HEAD do meu arquivo do navegador usando a tag <style>

@keyframes exemplo2 {
..0% {background-color: silver;}
..25% {background-color: royalblue;}
..50% {background-color: greenyellow;}
..75% {background-color: orange;}
..100% {background-color: rebeccapurple;}
}

.animacao2 { width: 150px; height: 150px; background-color: silver; animation-name: exemplo2; animation-duration: 5s; animation-delay: 2s; animation-iteration-count: infinite; }

No BODY do meu arquivo do navegador

<div class="animacao2"></div>

E esse código resulta nessa animação:

A propriedade animation-delay especifica um atraso para o início de uma animação. A animação anterior tem um delay de 2 segundos.

No exemplo a seguir, também mudamos a margem esquerda e superior em cada momento da animação.

No meu arquivo de estilos (style.css nesse caso)
OU
no HEAD do meu arquivo do navegador usando a tag <style>

@keyframes exemplo3 {
..0% {background-color: red; left: 0px; top: 0px;}
..25% {background-color: yellow; left: 50%; top: 0px;}
..50% {background-color: blue; left: 50%; top: 50%;}
..75% {background-color: green; left: 0px; top: 50%;}
..100% {background-color: red; left: 0px; top: 0px;}
}
.animacao3 {
..width: 150px;
..height: 150px;
..background-color: silver;
..animation-name: exemplo3;
..animation-duration: 5s;
..animation-delay: 2s;
..animation-iteration-count: infinite;
}


No BODY do meu arquivo do navegador

<div class="animacao3"></div>

A propriedade animation-iteration-count especifica o número de vezes que uma animação deve ser executada. Os exemplos acima usam o valor "infinito" na propriedade animation-iteration-count para fazer a animação continuar para sempre.

A propriedade animation-direction especifica se uma animação deve ser reproduzida para frente, para trás ou em ciclos alternados. Essa propriedade pode ter os seguintes valores:

  • normal - A animação é reproduzida normalmente (para frente). Este é o padrão.
  • reverse - A animação é reproduzida na direção reversa (para trás).
  • alternate - A animação é reproduzida primeiro para a frente, depois para trás.
  • alternate-reverse - A animação é reproduzida primeiro para trás e depois para a frente.

A propriedade animation-timing-function especifica a curva de velocidade da animação. Ela pode ter os seguintes valores:

  • ease - Especifica uma animação com um início lento, depois rápido e, a seguir, um final lento (este é o padrão).
  • linear - Especifica uma animação com a mesma velocidade do início ao fim.
  • ease-in - Especifica uma animação com um início lento.
  • ease-out - Especifica uma animação com um final lento.
  • ease-in-out - Especifica uma animação com início e fim lentos.
  • cubic-bezier (n,n,n,n). Permite definir seus próprios valores em uma função cúbica de bezier.
