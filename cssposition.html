CSS Position

A propriedade position especifica o tipo de método de posicionamento usado para um elemento. Existem cinco valores de posição diferentes:

  • static (estático)
  • relative (relativo)
  • fixed (fixo)
  • absolute (absoluto)
  • Sticky (aderente)

Os elementos são posicionados usando as propriedades top, bottom, left e right. No entanto, essas propriedades não funcionarão a menos que a propriedade position seja configurada primeiro. Eles também funcionam de forma diferente dependendo do valor da posição.

Position é o comportamento do elemento em relação a posição/posicionamento. Como ele vai se comportar: de forma estática, relativa, fixa, absoluta ou aderente. Top, bottom, left e right - se comportarão diferente de acordo com o valor da propriedade position.

Propriedade - position: static

Os elementos HTML são posicionados estáticos por padrão. Os elementos posicionados estáticos não são afetados pelas propriedades superior, inferior, esquerda e direita. Um elemento com postion static não está posicionado de forma especial, está sempre posicionado de acordo com o fluxo normal da página.

.posicaoEstatica{
..position: static;
..border: 3px solid #73AD21;
}

Esse elemento tem posição estática.

Propriedade: - position: relative

Um elemento com position relative está posicionado em relação à sua posição normal. Definir as propriedades top right bottom e left de um elemento relativamente posicionado fará com que ele seja ajustado para longe de sua posição normal. Outro conteúdo não será ajustado para caber em qualquer lacuna deixada pelo elemento.

.posicaoRelativa{
..position: static;
..left: 30px;
..border: 3px solid #73AD21;
}

Esse elemento tem posição relativa.

Propriedade: - position: fixed

Um elemento com position fixed está posicionado em relação à janela de visualização, o que significa que sempre permanece no mesmo lugar, mesmo se a página for rolada. As propriedades top right bottom e left serão usadas para posicionar o elemento. Um elemento fixo não deixa uma lacuna na página onde normalmente estaria localizado.

.posicaoFixada{
..position: fixed;
..bottom: 150px;
..right: 0;
..width: 300px;
..border: 3px solid #73AD21;
}

O elemento vai ficar grudado a direita da tela, aceite.

Esse elemento tem posição fixa.

Propriedade - position: absolute

Um elemento com position absolute é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à janela de visualização, como fixed. Contudo se um elemento posicionado de forma absoluta não tiver ancestrais posicionados, ele usará o corpo do documento e se moverá junto com a rolagem da página.

Nota: Um elemento absoluto é aquele cuja posição é qualquer uma, exceto estática.

.posicaoRelativaPaiAbsoluto{
..position: relative;
..width: 400px;
..height: 200px;
..border: 3px solid #73AD21;
}
.posicaoAbsoluta {
..position: absolute;
..top: 80px;
..right: 0;
..width: 200px;
..height: 100px;
..border: 3px solid #73AD21;
}

Esse elemento tem posição relativa - elemento pai do absoluto (nesse caso).
Esse elemento tem posição absoluta.

Propriedade - position: sticky

Um elemento com position sticky é posicionado com base na posição de rolagem do usuário. Um elemento "aderente" alterna entre "relativo" e "fixo" dependendo da posição de rolagem. Ele é posicionado em relação até que uma determinada posição de deslocamento seja encontrada na viewport - então, ele "se fixa" no lugar (como a posição fixa). Neste exemplo, o elemento aderente adere ao topo da página top 0 quando atingimos a posição de rolagem.

.posicaoAderente {
..position: sticky;
..top: 0;
..background-color: green;
..border: 2px solid #4CAF50;
}

Esse elemento tem posição aderente (sticky).

Elementos sobrepostos

Quando os elementos são posicionados, eles podem se sobrepor a outros elementos. A propriedade z-index especifica a ordem da pilha de um elemento (qual elemento deve ser colocado na frente ou atrás dos outros). Um elemento pode ter uma ordem de empilhamento positiva ou negativa.

  • bottom layer (mais distante do observador)
  • ...
  • Layer -3
  • Layer -2
  • Layer -1
  • Layer 0 (default rendering layer)
  • Layer 1
  • Layer 2
  • Layer 3
  • ...
  • top layer (mais próximo do observador)

Nota: Se dois elementos posicionados se sobrepõem sem um índice z especificado, o elemento posicionado por último no código HTML será mostrado na parte superior.

PORÉM: esse elemento vai depender do contexto que ele estiver inserido.

Código CSS (minha folha de estilos style.css)

.containerElemSobrepostos {
background-color: white;
width: 100%;
height: 440px;
position: relative;
align-content: center;
border: 1px dashed black;
}

.elementoSobreposto {
padding: 10px;
opacity: 0.7;
text-align: center;
}

#abs1 {
z-index: 5;

top: 10px;
left: 10px;

background-color: #fdd;

Um elemento com uma ordem de pilha maior estará sempre na frente de um elemento com uma ordem de pilha menor. Note que o elemento com z-index 8 está atrás dos outros elementos, mas é porque ele não está posicionado. Se definirmos a posição dele, ele irá sobrepor todos os outros.
