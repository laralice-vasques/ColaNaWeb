Hyperlinks HTML

A "url" pode ser um link externo e também pode ser um caminho/diretório.
Mas, quando o destino é um link da internet, deve-se levar em consideração que o conteúdo pode ser modificado ou o link alterado.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores

  • Um link não visitado está sublinhado e em azul
  • Um link visitado está sublinhado e roxo
  • Um link ativo está sublinhado e vermelho

Por padrão, a página vinculada será exibida na janela do navegador atual. Para alterar isso, você deve especificar outro destino para o link.

O atributo target especifica onde abrir o documento vinculado e ele pode ter um dos seguintes valores:

  • _self - Padrão Abre o documento na mesma janela/guia em que foi clicado;
  • _blank - Abre o documento em uma nova janela ou guia;
  • _parent - Abre o documento no quadro pai;
  • _top Abre o documento em todo o corpo da janela;

Código:

<a target="_blank" href="https://www.google.com/">Clicar aqui vai abrir o google em uma nova guia</a>

Navegador:

Clicar aqui irá abrir o Google em uma nova guia

Também é possível usar uma imagem como um link. No exemplo abaixo, clicar na imagem irá te enviar para a página principal do Google.

Código:

<a href ="https://www.google.com.br/"> <img src ="img/google.png" alt="HTML tutorial" style ="width:42px;height:42px;"> </a>

O link em CSS

a:link {}
Define a aparência dos links ainda não visitados.

a:visited {}
Define a aparência dos links já visitados.

a:hover {}
Define a aparência dos links quando o usuário passa o mouse sobre eles.

a:focus {}
Define a aparência dos links quando o usuário clica em um link.

a:active {}
Define a aparência dos links ativos, por exemplo, quando navegamos entre os links pelo teclado utilizando a tecla Tab.
