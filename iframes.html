Iframes HTML

Um iframe HTML é usado para exibir uma página da web dentro de uma página da web. A tag HTML <iframe> especifica um quadro embutido - que é usado para incorporar outro documento ao documento HTML atual.

Sintaxe: <iframe src="url" title="description">

Na minha página HTML

<iframe
..src="https://newyorkmais.com/"
..height="350"
..width="600"
..style="border: none"
..title="Exemplo de um Iframe - Meus conteúdos NY+">
</iframe>
