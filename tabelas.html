Tabelas HTML

A tag <table> define uma tabela HTML.

  • Cada linha da tabela é definida com uma tag <tr>;
  • Cada cabeçalho da tabela é definido com uma tag <th>;
  • Cada dado/célula da tabela é definido com uma tag <td>;
  • Por padrão, o texto nos elementos <th> está em negrito e centralizado;
  • Por padrão, o texto nos elementos <td> são regulares e alinhados à esquerda.
<table>
..<tr>
....<th>Cabeçalho-Conteúdo1</th>
....<th>Cabeçalho-Conteúdo2</th>
....<th>Cabeçalho-Conteúdo3</th>
..</tr>
..<tr>
....<td>Dado/célula-Conteúdo1</td>
....<td>Dado/célula-Conteúdo2</td>
....<td>Dado/célula-Conteúdo3</td>
..</tr>
..<tr>
....<td>Outro dado/célula-Conteúdo1</td>
....<td>Outro dado/célula-Conteúdo2</td>
....<td>Outro dado/célula-Conteúdo3</td>
..</tr>
</table>

O código da tabela acima é esta tabela (com estilo na folha externa margin: auto; padding: 10px; border: 1px solid black;):

Cabeçalho-Conteúdo1 Cabeçalho-Conteúdo2 Cabeçalho-Conteúdo3
Dado/célula-Conteúdo1 Dado/célula-Conteúdo2 Dado/célula-Conteúdo3
Outro dado/célula-Conteúdo1 Outro dado/célula-Conteúdo2 Outro dado/célula-Conteúdo3

O cabeçalho também pode ficar do lado, é só trocar o th e o td:

<table>
..<tr>
....<th>Cabeçalho1</th>
....<td>Dado/célula-Conteúdo1</td>
....<td>Outro dado/célula-Conteúdo1</td>
..</tr>
..<tr>
....<th>Cabeçalho2</th>
....<td>Dado/célula-Conteúdo2</td>
....<td>Outro dado/célula-Conteúdo2</td>
..</tr>
..<tr>
....<th>Cabeçalho3</th>
....<td>Dado/célula-Conteúdo3</td>
....<td>Outro dado/célula-Conteúdo3</td>
..</tr>
</table>

O código acima é a tabela abaixo (com estilo na folha externa margin: auto; padding: 10px; border: 1px solid black;)

Cabeçalho1 Dado/célula-Conteúdo1 Outro dado/célula-Conteúdo1
Cabeçalho2 Dado/célula-Conteúdo2 Outro dado/célula-Conteúdo2
Cabeçalho3 Dado/célula-Conteúdo3 Outro dado/célula-Conteúdo3
