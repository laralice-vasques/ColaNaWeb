Introdução a HTML

HTML (Hyper Text Markup Language) é a linguagem de marcação padrão para páginas da web. HTML consiste em uma série de elementos e atributos. Os elementos HTML informam ao navegador como exibir o conteúdo e identificam partes de conteúdo como "isto é um título", "isto é um parágrafo", "isto é um link", etc.

O que é um elemento HTML?

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final:

Tag inicial Conteúdo do elemento Tag final
<h1> Primeiro cabeçalho </h1>
<p> Primeiro parágrafo </p>
<br> none none

Tag de abertura - Consiste no nome do elemento, envolvido entre parênteses angulares de abertura e fechamento. Isso indica onde o elemento começa, ou inicia a produzir efeito. Neste caso, onde o parágrafo se inicia.

Tag de fechamento - É o mesmo que a tag de abertura, exceto que este inclui uma barra diagonal antes do nome do elemento. Indica onde o elemento termina, neste caso, onde fica o fim do parágrafo.

O conteúdo - Este é o conteúdo do elemento, que neste caso é somente texto.

O elemento - A tag de abertura, mais a tag de fechamento, mais o conteúdo, é igual ao elemento.

O objetivo de um navegador da web (Chrome, Edge, Firefox, Safari) é ler documentos HTML e exibi los corretamente. Um navegador não exibe as tags HTML, mas as usa para determinar como exibir o documento:


Tag: <b> conteúdo </b>

Código: Nunca foi sorte, sempre foi <b>Deus</b>!

Navegador: Nunca foi sorte, sempre foi Deus!


Tag: <strong> conteúdo </strong>

Código: Nunca foi sorte, sempre foi <strong>Deus</strong>!

Navegador: Nunca foi sorte, sempre foi Deus!


Tag: <i> conteúdo </i>

Código: Nunca foi sorte, sempre foi <i>Deus!</i>

Navegador: Nunca foi sorte, sempre foi Deus!


Tag: <em> conteúdo </em>

Código: Nunca foi sorte, sempre foi <em>Deus</em>!

Navegador: Nunca foi sorte, sempre foi Deus!


Tag: <u> conteúdo </u>

Código: Nunca foi sorte, sempre foi <u>Deus</u>!

Navegador: Nunca foi sorte, sempre foi Deus!


Tag: <strike> conteúdo </strike>

Código: Nunca foi <strike>sorte</strike>, sempre foi Deus!

Navegador: Nunca foi sorte, sempre foi Deus!

Anatomia de uma Tag HTML

Elementos também podem conter atributos, que se apresentam da seguinte forma:


  • Todos os elementos HTML podem ter atributos;
  • Os atributos fornecem informações adicionais sobre os elementos;
  • Os atributos são sempre especificados na tag inicial;
  • Os atributos geralmente vêm em pares nome/valor como nome="valor".

Um atributo deve conter

  • Um espaço entre ele e o nome do elemento (ou o atributo anterior, caso o elemento já contenha um ou mais atributos);
  • O nome do atributo seguido por um sinal de igual;
  • Um valor de atributo com aspas de abertura e fechamento em volta dele.

Atributo style

Usado para adicionar estilos a um elemento, como: cor do texto, cor de fundo, fonte, tamanho etc... <tagname style="propriedade: valor;">

Propriedade color:

Código:

<p style ="color:blue;"> Isto é um parágrafo azul!</p>

Navegador:

Isto é um parágrafo azul!

Propriedade background-color:

Código:

<p style ="background-color:silver;">Este parágrafo tem fundo cinza</p>

Navegador:

Este parágrafo tem fundo cinza

Propriedade font-family:

Código:

<p style ="font-family:verdana;">Parágrafo de fonte verdana</p>

Navegador:

Parágrafo de fonte verdana

Código:

<p style ="font-family:courier;">Parágrafo de fonte courier</p>

Navegador:

Parágrafo de fonte courier

Propriedade font-size:

Código:

<p style ="font-size:80%;">Parágrafo tamanho 80%</p>

Navegador:

Parágrafo tamanho 80%

Código:

<p style ="font-size:130%;">Parágrafo tamanho 130%</p>

Navegador:

Parágrafo tamanho 130%

Propriedade text-align:

Código:

<p style="text-align: left;">Esquerda</p>
<p style="text-align: center;">Centro</p>
<p style="text-align: right;">Direita</p>

Navegador:

Esquerda

Centro

Direita

Atributo title

Define algumas informações extras sobre um elemento. O valor do atributo title será exibido como uma dica (tooltip) quando passarmos o mouse sobre o elemento

Código:

<p title ="Eu sou uma dica!"> Isto é um parágrafo.</p>

Navegador:

Isto é um parágrafo.

Atributo lang

Deve ser sempre incluido como atributo tag <html>, para declarar o idioma da página web. Isso serve para auxiliar os mecanismos de pesquisa e navegadores. O exemplo a seguir especifica o inglês como o idioma

Código:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

classe em JavaScript

O nome da classe também pode ser usado pelo JavaScript para realizar certas tarefas para elementos específicos. O JavaScript pode acessar elementos com um nome de classe específico com o método: getElementsByClassName()

Código:

<button onclick="myFunction()">Botão mágico</button>

<script>
..function myFunction() {
....var x = document.getElementsByClassName("exemploAtributoClasseJS");
....for (var i = 0; i < x.length; i++){
......x[i].style.display ="none";
....}
}
</script>

<div class="exemploAtributoClasseJS">
..Essa div vai sumir assim que pressionar o botão mágico
</div>

Navegador:

Essa div vai sumir assim que pressionar o botão mágico

id em JavaScript

O atributo id também pode ser usado pelo JavaScript para realizar algumas tarefas para aquele elemento específico. O JavaScript pode acessar um elemento com um id específico com o método getElementById()

Código:

<p id="botaoMandao">Clique no botão abaixo!</p>
<button onclick="displayResult()">Esse aqui</button>
<script>
..function displayResult() {
....document.getElementById("botaoMandao").innerHTML ="Obrigada, era só isso.";
}
</script>

Navegador:

Clique no botão abaixo!
