Tranformações 2D

Tranformações CSS 2D

As transformações CSS permitem mover, girar, dimensionar e inclinar elementos.

Propriedade transform e métodos de transformação: com a propriedade transform podemos usar os seguintes métodos de transformação 2D:

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()

Metódo: translate()

O método translate() move um elemento de sua posição atual de acordo com os parâmetros fornecidos para os eixos X e Y.

O exemplo a seguir move o elemento 50 pixels para a direita e 70 pixels para baixo de sua posição atual.

<p>Imagem original</p>
<img src ="img/borboleta2.jpg " width="200px">

<p>Imagem transformada</p>
<img src ="img/borboleta2.jpg "
width="200px"
style="transform: translate(50px, 70px);">

Método: rotate()

O método rotate() gira um elemento no sentido horário ou anti horário de acordo com um determinado grau.
Usar valores negativos girará o elemento no sentido anti horário.

O exemplo a seguir gira o elemento no sentido horário em 20 graus.

<p>Imagem original</p>
<img src ="img/borboleta2.jpg " width="200px">

<p>Imagem transformada</p>
<img src ="img/borboleta2.jpg "
width="200px"
style="transform: rotate(20deg);">

Método: scale()

  • O método scale() aumenta ou diminui o tamanho de um elemento de acordo com os parâmetros dados para largura e altura.
  • O método scaleX() aumenta ou diminui a largura de um elemento.
  • O método scaleY() aumenta ou diminui a altura de um elemento.

O exemplo a seguir aumenta o elemento para 1.3 vezes de sua largura original e 1.4 vezes de sua altura original - scale().

<p>Imagem original</p>
<img src ="img/borboleta2.jpg " width="200px">

<p>Imagem transformada</p>
<img src ="img/borboleta2.jpg "
width="200px"
style="transform: scale(1.3, 1.4);">

Método: matrix()

O método matrix() combina todos os métodos de transformação 2D em um.
Ele usa seis parâmetros, contendo funções matemáticas, que permitem girar, dimensionar, mover (traduzir) e inclinar elementos.

Os parâmetros são os seguintes:
matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

<p>Imagem original</p>
<img src ="img/borboleta2.jpg " width="200px">

<p>Imagem transformada</p>
<img src ="img/borboleta2.jpg "
width="200px"
style="transform: matrix(1, -0.3, 0, 1, 0, 0);">

Transições

Transições CSS

As transições CSS permitem que você altere os valores das propriedades suavemente, ao longo de um determinado período.

Propriedades listadas aqui:

  • transition
  • transition-delay
  • transition-duration
  • transition-property
  • transition-timing-function

Para criar um efeito de transição, devemos especificar duas coisas:

  1. a propriedade CSS à qual desejamos adicionar um efeito;
  2. a duração do efeito.

Se a parte da duração não for especificada, a transição não terá efeito, porque o valor padrão é 0.

div {
. width: 100px;
. height: 100px;
. background: red;
. transition: width 2s;
}

O efeito de transição começará quando a propriedade CSS especificada (largura) alterar o valor. Agora, vamos especificar um novo valor para a propriedade de largura quando o usuário passar o mouse sobre o elemento <div>:

No meu .css

.transicaoEx1 {
width: 100px;
height: 100px;
background: royalblue;
transition: width 2s;
}

.transicaoEx1:hover {
width: 300px;
}


No meu .html

<div class="transicaoEx1"></div>



Curva de velocidade da transição

A propriedade transition timing function especifica a curva de velocidade do efeito de transição. A propriedade da função de tempo de transição pode ter os seguintes valores:

  • ease - especifica um efeito de transição com um início lento, depois rápido e, em seguida, termina lentamente (este é o padrão);
  • linear - especifica um efeito de transição com a mesma velocidade do início ao fim;
  • ease-in - especifica um efeito de transição com início lento;
  • ease-out - especifica um efeito de transição com atenuação de fim lento;
  • ease-in-out especifica um efeito de transição com um início e fim lentos;
  • cubic-bezier(n, n, n, n) permite definir seus próprios valores em uma função cúbica debezier.

Atraso no efeito de transição

A propriedade transition-delay especifica um atraso (em segundos) para o efeito de transição. O exemplo a seguir tem um atraso de 1 segundo antes de começar:

No meu .css

.transicaoEx2 {
width: 100px;
height: 100px;
background: royalblue;
transition: width 2s;
transition-delay: 1s;
}

.transicaoEx2:hover {
width: 300px;
}

No meu .html

<div class="transicaoEx2"></div>

Transição + Transformação

O exemplo a seguir adiciona um efeito de transição à transformação:

No meu .css

.transicaoEx3 {
width: 100px;
height: 100px;
background: royalblue;
transition: width 2s, height 2s, transform 2s;
}

.transicaoEx3:hover {
width: 300px;
height: 300px;
transform: rotate(180deg);
}

No meu .html

<div class="transicaoEx3"></div>
